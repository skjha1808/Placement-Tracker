# Placement Tracker

An AI-powered MERN stack application to help students manage placement applications, interview rounds, deadlines, and job offers efficiently.

## Features

### Authentication
- Secure Login & Signup
- JWT Authentication
- Role-based Access Control

### Student Management
- Student Profile
- Skills and CGPA Tracking

### Company Management
- Company Details
- Eligibility Criteria
- Application Deadlines

### Application Tracking
- Applied
- OA Scheduled
- OA Cleared
- Interview Scheduled
- Selected
- Rejected

### Dashboard
- Total Applications
- Active Processes
- Success Rate
- Placement Statistics

### AI Features
- Resume Analyzer
- Interview Question Generator
- Job Description Analyzer

## Tech Stack

**Frontend:** React.js, React Router, Axios

**Backend:** Node.js, Express.js

**Database:** MongoDB, Mongoose

**Authentication:** JWT, bcrypt

**AI:** OpenAI API
